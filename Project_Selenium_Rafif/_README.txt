WHATSAPP AUTO-REPLY BOT SYSTEM
==============================

üìå DESKRIPSI
-------------
Program ini adalah sistem otomatisasi berbasis Python yang terhubung dengan WhatsApp Web melalui Selenium.
Bot akan:
- Mendeteksi pesan masuk dari kontak tertentu
- Mencatat isi pesan ke database SQLite
- Memberikan balasan otomatis

üìÅ STRUKTUR FOLDER
-------------------
1. AutoconnectWhatsapp/
   - Script untuk menyambungkan otomatis ke WhatsApp Web

2. Gambar/
   - Script untuk mengirim gambar

3. Interaktif/
   - Script bot interaktif berbasis keyword (misal: #kode)

4. LogsWhatsapp/
   - Script untuk mencatat log yang masuk di WhatsApp(Harian)

5. MySQL/
   - Script untuk mencatat log yang masuk di WhatsApp kemudian ke mySQL

6. Automation.py
   - Script utama yang menjalankan bot WhatsApp dan logging ke SQLite

7. chromedriver.exe
   - Driver yang digunakan Selenium untuk membuka Chrome browser
